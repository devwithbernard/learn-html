HTML Horizontal Rules

The < hr /> tag defines a thematic break in an HTML page, and is most often displayed as a Horizontal rule.

The < hr /> element is used to separate content (or define a change )in a HTML page .

HTML Line Breaks

The HTML < br /> element defines a line break.

Use < br /> is you want a line break (a new line) without starting a new paragraph.
